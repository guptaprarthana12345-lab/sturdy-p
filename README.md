# EduQuest 🎮

EduQuest is a gamified habit tracker built with Next.js.  
It combines **classic arcade style UI** with productivity features like habits, goals, XP, streaks, rewards, and a companion chat assistant.

## ✨ Features
- 🎯 **Habits**: Add, complete, and track streaks (+10 XP each).
- 🏆 **Goals**: One‑time achievements (+30 XP).
- 📊 **Stats**: Weekly and monthly progress with ✔ and ✘ indicators.
- 🎁 **Rewards**: Unlock and redeem rewards using XP.
- 🦊🦉 **Companion**: Choose Fox or Owl avatar, chat with them, get suggestions or auto‑decisions.
- 🌟 **Arcade UI**: Pixel‑style cards, glowing XP bars, retro sidebar.

## 🛠️ Tech Stack
- [Next.js](https://nextjs.org/) (React framework)
- TypeScript
- CSS (custom arcade theme)
- LocalStorage (persistent data)
